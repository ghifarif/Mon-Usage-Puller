This script automatically pull utilization csv data to be displayed in analytical platform (Redash localhost in this case). Need orchestration such as jenkins/circleCI for schedule pulling automatically.
Can be used for similiar case to other monitoring tool, provided API is supported.

Refference used/related in this project:
- [Zabbix API](https://www.zabbix.com/documentation/current/en/manual/api)
- [Datadog API](https://docs.datadoghq.com/api/latest/)
